# Homelab

A small support repository for self-hosted dashboard assets.

This is not a full homelab deployment repository. It is a practical asset repository intended for referencing assets.

# Licence

Unless otherwise restricted by third-party rights, this repository is distributed under the terms set out in LICENSE.md.

Because some assets refer to third-party brands and marks, licence and trademark considerations may differ between the repository contents and the underlying brand assets themselves.

# Intended use

This repository is primarily for:

- private homelab dashboards
- self-hosted service portals
- local asset mirroring for Homer
- maintaining a reusable icon pool for personal infrastructure projects

It is **not** presented as an official source for any third-party brand asset.
